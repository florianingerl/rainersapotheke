1) To run the backend
1.1) cd backend
1.2) npm install
1.3) node server.js
1.4) open https://localhost:3000 in your browser

2) To run the frontend
2.1) First start the server for the backend
2.2) cd frontend
2.3) npm install
2.4) npm run serve
2.5) open https://localhost:8080 in your browser
